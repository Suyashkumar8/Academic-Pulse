# Academic Pulse

Academic Pulse is a C++ console-based application designed to help students monitor and analyze their academic performance and coding skill development.

The application collects information about subjects, attendance, marks, and programming skills, then generates a structured performance report with useful insights and recommendations.

---

## Features

- Student profile management
- Dynamic subject and coding skill input
- Automatic grade calculation (A+, A, B, C, F)
- Subject ranking based on marks
- Coding skill ranking based on hours invested
- Attendance analysis with low-attendance alerts
- Academic performance evaluation
- Target achievement analysis
- Coding skill progress visualization using progress bars
- Generates a detailed report in `Academic_Report.txt`

---

## Technologies Used

- C++
- Object-oriented programming concepts using structures
- Functions
- Arrays
- File handling (`ofstream`)
- Sorting algorithms
- Console-based user interface
